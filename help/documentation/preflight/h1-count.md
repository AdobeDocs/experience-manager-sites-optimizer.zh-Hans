# 预检 H1 计数机会

![预检 H1 计数机会](./assets/h1-count/hero.png){align="center"}

Sites Optimizer 中的预检 H1 计数机会可帮助您识别并解决网站中与使用 H1 标题相关的问题。该机会重点在于评估网站的结构，并提供可操作的改进建议。
