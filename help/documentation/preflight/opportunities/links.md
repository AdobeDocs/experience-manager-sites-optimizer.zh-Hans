# 预检链接机会

![预检链接机会](./assets/links/hero.png){align="center"}

Sites Optimizer 中的预检链接机会可帮助您识别并解决网站中与使用链接相关的问题。 该机会重点在于评估网站的链接结构，并提供可操作的改进建议。
