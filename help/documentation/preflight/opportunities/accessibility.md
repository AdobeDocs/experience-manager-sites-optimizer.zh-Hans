# 预检辅助功能机会

![预检辅助功能机会](./assets/accessibility/hero.png){align="center"}

Sites Optimizer中的“预检辅助功能”机会可帮助您识别并解决网站上的辅助功能问题，以确保包括残障人士在内的所有个人都可以使用。 该机会侧重于评估网站对辅助功能标准的符合程度，并提供可操作的改进建议。
