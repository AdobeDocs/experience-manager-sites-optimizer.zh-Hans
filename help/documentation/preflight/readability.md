# 预检可读性机会

![预检可读性机会](./assets/readability/hero.png){align="center"}

Sites Optimizer 中的预检可读性机会可帮助您识别和处理与网站内容可读性相关的问题。该机会重点在于评估网站的内容结构，并提供可操作的改进建议。

预检可读性机会提供多种语言，包括：

* 英语
* 法语
* 西班牙语
* 意大利语
* 德语
* 荷兰语

它确保团队可以评估和优化各种受众中的内容可读性。
