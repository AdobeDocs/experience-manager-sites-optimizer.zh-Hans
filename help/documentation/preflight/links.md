# 印前检查链接机会

![印前检查链接机会](./assets/links/hero.png){align="center"}

Sites Optimizer中的“预检链接”机会可帮助您识别并处理与网站上链接的使用相关的问题。 此机会重点评估您网站的链接结构，并提供切实可行的改进建议。
