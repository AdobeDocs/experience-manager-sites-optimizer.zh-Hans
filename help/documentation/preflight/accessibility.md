# 预检辅助功能机会

![预检辅助功能机会](./assets/accessibility/hero.png){align="center"}

Sites Optimizer中的预检辅助功能机会可帮助您识别并解决网站上的辅助功能问题，以确保包括残障人士在内的所有个人都可以使用。 此机会重点评估您站点对辅助功能标准的符合性，并提供切实可行的改进建议。
