# 印前检查元数据机会

![预检元数据机会](./assets/metadata/hero.png){align="center"}

Sites Optimizer中的预检元数据机会可帮助您识别并解决与在网站上使用元数据相关的问题。 此机会重点评估您网站的元数据结构，并提供切实可行的改进建议。
